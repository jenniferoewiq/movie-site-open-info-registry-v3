# movie-site-open-info-registry-v3
本仓库用于登记影视类网站的公开基础信息，对互联网上已公开存在的电影网站进行系统化记录。整理内容涵盖站点名称、访问地址、站点定位说明及基础分类结构，便于集中管理与长期保存。本项目不涉及影视资源播放、下载或解析，仅作为信息档案存在。所有信息整理自公开网络环境，不构成任何形式的推荐或使用引导。
